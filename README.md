## Requirements

Please ensure that all the libraries below are successfully installed:

* python==3.8.19
* torch==2.4.1
* torch_geometric==2.5.3
* pandas==1.3.5
* numpy==1.22.4
* scikit-learn==1.3.2

## Usage

You can directly run the following code to run the model:

```
cd ./Code/
python main.py
```
